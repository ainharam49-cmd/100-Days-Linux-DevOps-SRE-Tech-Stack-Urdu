# Task 1 - Repository Status Check Karna

## Objective

Apni Git repository ka current status check karna.

---

## Run:

```bash
git status
```

---

## Observation

Check karein:

- Kaunsi files modified hain
- Kaunsi files untracked hain
- Kaunsi files commit ke liye ready hain

---

# Task 2 - Tamam Files Stage Karna

## Objective

Repository ki tamam files ko stage karna.

---

## Run:

```bash
git add .
```

---

## Explanation

Yeh command stage karti hai:

- README files
- Images
- Markdown documents
- Tamam modified files

next commit ke liye.

---

# Task 3 - Changes Commit Karna

## Objective

Apni changes ko Git history mein save karna.

---

## Run:

```bash
git commit -m "README file updated"
```

---

## Explanation

- `commit` aapka kaam save karta hai
- `-m` commit message add karne ke liye use hota hai

---

# Task 4 - GitHub par Code Push Karna

## Objective

Apni local changes ko GitHub par upload karna.

---

## Run:

```bash
git push
```

---

## Explanation

Yeh command aapki committed files ko:

```text
Local Repository
```

se:

```text
Remote Repository (GitHub)
```

par bhejti hai.

---

# Task 5 - GitHub par Push Verify Karna

## Objective

Confirm karna ke aapki files successfully upload ho gayi hain.

---

## Instructions

1. Apni GitHub Repository open karein
2. Page refresh karein
3. Verify karein ke:
   - README files nazar aa rahi hain
   - Images upload ho gayi hain
   - Markdown files visible hain
   - Latest commits show ho rahe hain

---

## IMPORTANT NOTE

Code push karne ke baad GitHub page refresh karna mat bhooliye.
