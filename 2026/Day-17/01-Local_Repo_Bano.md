## Important Concepts

### Repository = Repo = Directory + Files

Repository (Repo) mein shamil hotay hain:

- Directories (Folders)
- Files
  - Images
  - Documents
  - Markdown Files
  - Source Code
  - Videos
  - Dusri Files

---

## Local Repository (Local Repo)

Jab hum apne Laptop/Desktop par directories aur files create karte hain to isay kehte hain:

```text
Local Repository (Local Repo)
```

## Remote Repository (Remote Repo)

Jab hum apne account se:

```text
github.com
```

par ek nayi repository create karte hain to isay kehte hain:

```text
Remote Repository (Remote Repo)
```
