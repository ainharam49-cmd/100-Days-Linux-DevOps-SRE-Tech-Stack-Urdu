# Task 1 - SSH Keys Generate Karna

## Objective

GitHub authentication ke liye SSH Public aur Private Keys generate karna.

---

## Neeche Di Gayi Command Run Karein

```bash
ssh-keygen -t rsa -b 4096 -C "your_email@example.com"
```

> NOTE: `"your_email@example.com"` ko apne actual email address se replace karein.

---

## Example

```bash
ssh-keygen -t rsa -b 4096 -C "nadeem.siddiqi@nit.academy"
```

---

## Apna Git Username Verify Karein

Run:

```bash
git config user.name
```

## Example Output

```bash
mike.mill
```

---

# Task 2 - SSH Agent Start Karna

## Objective

SSH authentication agent ko background mein start karna.

---

## Run:

```bash
eval "$(ssh-agent -s)"
```

## Example Output

```bash
Agent pid 459
```

---

# Task 3 - SSH Private Key Add Karna

## Objective

Apni SSH private key ko SSH Agent mein add karna.

---

## Run:

```bash
ssh-add ~/.ssh/id_rsa
```

## Example Output

```bash
Identity added: /c/Users/nsidd/.ssh/id_rsa (mike.mill@adptech.com)
```

---

## IMPORTANT NOTE

Agar aapki SSH private key ka naam different hai jaise:

```bash
id_ed25519
```

to `id_rsa` ki jagah apna actual key filename use karein.

Example:

```bash
ssh-add ~/.ssh/id_ed25519
```

---

# Task 4 - SSH Public Key Copy Karna

## Objective

SSH Public Key ko copy karna taake usay GitHub mein add kiya ja sake.

---

## Run:

```bash
clip < ~/.ssh/id_rsa.pub
```

---

## Next Steps

1. **Notepad** open karein
2. Keyboard par press karein:

```text
Ctrl + V
```

3. Verify karein ke SSH Public Key successfully NOTEPAD mein paste ho gayi hai.

# Task 5 - GitHub mein SSH Key Add Karna

## Objective

Apni SSH Public Key ko GitHub account mein add karna.

---

## Neeche Diye Gaye Steps Follow Karein

### Step 1

GitHub mein Top Right Profile Icon par click karein.

---

### Step 2

Select karein:

```text
Settings
```

---

### Step 3

Click karein:

```text
SSH and GPG Keys
```

---

### Step 4

Click karein:

```text
New SSH Key
```
