# Task 1 - Git Username Update Karna

## Objective

Apna Git username configure karein.

---

## Run:

```bash

```

---

## Explanation

Yeh command aapka Git username globally update karti hai aapki local machine par.

# Task 2 - Git User Email Update Karna

## Objective

# Task 9 - Remote GitHub Repository Add Karna

```

---

## IMPORTANT NOTE

```

---

# Task 10 - GitHub par Code Push Karna

## Objective

Local code ko Remote GitHub Repository par push karna.

---

## Run:

```bash
git push -u origin master
```

---

## Explanation

| Command Part | Purpose |
|--------------|----------|
| git push | Code ko remote repository par bhejta hai |
| -u | Upstream tracking set karta hai |
| origin | Remote repository ka naam |
| master | Branch ka naam |

---

## IMPORTANT NOTE

Agar aapki branch ka naam `main` hai to yeh command use karein:

```bash
git push -u origin main
```
